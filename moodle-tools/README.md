# moodle-tools
Minimal image with the tools that are necessary for our Moodle Kubernetes CronJobs.

## Provided Tools
* [wget](https://www.gnu.org/software/wget/)
* [kubectl](https://kubernetes.io/docs/reference/kubectl/)
* [boto](https://boto3.amazonaws.com/v1/documentation/api/latest/index.html)
* [curl](https://curl.se/)
* [duply](https://duply.net/)
* [ca-certificates](https://launchpad.net/ubuntu/+source/ca-certificates)
* [gnupg](https://gnupg.org/)
* [apt-transport-https](https://manpages.ubuntu.com/manpages/jammy/en/man1/apt-transport-https.1.html)
* [psql](https://www.postgresql.org/docs/current/app-psql.html)
* [mariadb](https://mariadb.com/docs/server/connect/clients/mariadb-client/)
